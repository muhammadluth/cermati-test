## Build Setup

1. Clone repository
   `$ git clone https://github.com/muhammadluth/cermati-test.git`

2. Install depedencies

```bash
# with npm
$ npm install

# or with yarn
$ yarn install
```

3. Start server

```bash
# with npm
$ npm start # start server

# or with yarn
$ yarn start # start server

```
